# prestashop-color-and-size
Prestashop. Hide out of stock sizes in current color

1) Downlad project files.
2) Open http://contentbox.org/ and create new module, for examle "Color And Size". Download it.
3) Install your module and go to configure it.
4) In "Content Configuration" section upload HideUnavailableSizes.js from this project. And click save in this section/
5) In "Developer Configurations" section set "Use Text Editor" - NO, and in "Load Files on HTML Header" select "HideUnavailableSizes"
